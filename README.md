Pet-проект по курсу https://stepik.org/course/180000/syllabus

В проект существует ролевая модель
-
- Buyer
- Seller
- Admin

Сущности проекта
-
- Категория
    - 
    - Создание категории (admin)
    - Удаление категории (admin)
    - Обновление категории (admin)
    - Просмотр одной или всех категорий (all)
- Продукт
    -
    - Создание продукта (seller)
    - Удаление продукта (seller)
    - Обновление продукта (seller)
    - Просмотр одного или всех продуктов (all)
- Отзыв
    - 
    - Создание отзыва (buyer)
    - Удаление отзыва (admin)
    - Просмотр одного или всех отзывов (all)
